## 1. Movie Recommendation System 🎬
A Machine Learning project designed to predict and recommend movies to users based on historical data, metadata, and user preferences.

### ✨ Features
* **Hybrid Filtering:** Implements Content-Based Filtering (matching genres, directors, and keywords) and Collaborative Filtering features.
* **Vector Similarity:** Utilizes **Cosine Similarity** to compute similarity scores between different movies for accurate recommendations.
* **Efficient Data Pipeline:** Cleaned and preprocessed raw datasets to handle missing values and filter out anomalies.

### 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Scikit-learn, Pandas, NumPy
* **Dataset:** TMDB / MovieLens

## 2. Tic-Tac-Toe AI 🎮
An interactive game application featuring an unbeatable Artificial Intelligence opponent driven by classic game theory algorithms.

### ✨ Features
* **Unbeatable AI Opponent:** Powered by the **Minimax Algorithm** to evaluate all possible board states and make optimal moves.
* **Alpha-Beta Pruning (Optional):** Optimized decision trees to reduce computation time and enhance game performance.
* **Clean Interface:** Provides a simple, bug-free grid layout for a smooth user-versus-computer gaming experience.

### 🛠️ Tech Stack
* **Language:** Python / JavaScript
* **GUI/Logic:** Tkinter / Pygame (or HTML/CSS for web version)

## 3. AI Chatbot 🤖
An intelligent conversational assistant engineered to process natural language inputs, recognize user intents, and deliver context-aware responses.

### ✨ Features
* **Intent Classification:** Uses NLP techniques for tokenization, stemming/lemmatization, and matching user queries to responses.
* **Context Preservation:** Structured to maintain conversational state across multi-turn user dialogues.
* **Modular Design:** Built with flexible routing, making it highly customizable for customer support or personal assistance.

### 🛠️ Tech Stack
* **Language:** Python
* **NLP Frameworks:** NLTK / Spacy / ChatterBot
* **Interface:** Command Line Interface (CLI) / Flask Web App
